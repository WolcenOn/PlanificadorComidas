Formato: código | marca | cantidad del envase | unidad | precio. Un producto por línea.

Actualizar stock rápido

Escanea o escribe un código de barras. Si ya está asociado a un ingrediente, se detecta automáticamente. Para alimentos a granel, usa la selección manual.

La cámara necesita permiso del navegador y normalmente requiere abrir la página desde HTTPS o localhost. Si no funciona, usa la entrada manual.
Para activar cámara en ordenador: abre una terminal en la carpeta del archivo y ejecuta python -m http.server 8000. Después entra en http://localhost:8000/gestor_comidas_semanas_codigos.html.

Biblioteca de packs oficial

La app solo lee archivos JSON de la carpeta packs del repositorio oficial WolcenOn/PlanificadorComidas. No se muestran campos para otros usuarios ni otros repositorios, así mantienes la calidad de los packs.

Repositorio bloqueado:
Repositorio oficial bloqueado: WolcenOn/PlanificadorComidas · carpeta /packs.
Estructura recomendada: packs/recetas para recetas y packs/ingredientes para listas de ingredientes. Se escanean también subcarpetas dentro de cada categoría.
Pulsa “Escanear biblioteca oficial” para leer los JSON publicados en /packs.
